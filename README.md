# *Code for: Spatial patterns of fire severity and bark beetle outbreak in the Cameron Peak Fire* 

## Description of the project

This repository contains all code for comparing patterns of burn severity and MPB outbreak across the Cameron Peak fire.

The project has the following structure:

-   *.gitignore* : A file specifying which files and directories should be ignored by Git version control.

-   *CITATION.cff* : A file specifying how to cite this project, including the authors and title. 

-   Code: This subdirectory contains all code written for this project. 

    -   *CBIxBB.Rmd* : This file contains code for downloading the public data, processing it, and generating figures.

    -   *global-change-biology.csl*: A csl file for formatting references in the style of the journal Global Change Biology.

    -   *references.bib*: A BibTeX file containing all information for all references used in the project

-   Results: This subdirectory contains results generated

    -   Figures: This subdirectory contains any figures generated

-   *License.md* : A file specifying the license under which this project is released, including any terms and conditions for use and distribution.

-   *README.md* : This file, which provides an overview of the project, its structure, and how to use it.

-   *FireSeverity-CameronPeak.Rproj* : An R project file for this project, which can be opened in RStudio to set the working directory and load any necessary packages.
